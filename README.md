## Instalando dependências

As dependências usadas no projeto são:
* SFML

Para instalar essas depêndencias do projeto rode no terminal o comando:
    
    sudo apt install libsfml-dev

## Rodando o projeto

Para executar o projeto use o comando:
    
    make all
